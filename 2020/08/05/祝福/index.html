祝福

发表于 2020-08-05
本文字数： 389 阅读时长 ≈ 1 分钟

背景

《祝福》是80年代末香港流行音乐中的经典之作。 这首歌曲由 梁宏志 作曲， 潘伟源 作词， 收录在歌手 叶倩文 的同名专辑《祝福》中， 由华纳唱片于1988年发行。

资料来源: 百度百科

评论

《祝福》的曲调细腻委婉， 歌词清丽动人， 用粤语演唱更是点睛之笔。 歌手叶倩文的演绎气定神闲， 随着她的歌声不妨去看看 “崇山峻岭， 茂林修竹”, 去听听 “清流激湍，映带左右”, 若是 “行到水穷处”, 那就 "坐看云起时"吧.
